About This Application

This is a demo hotel management and booking system built by Aman Verma to showcase my skills as a Full Stack Developer specializing in Angular + Java (Spring Boot).

Frontend Stack

  • Built with Angular 20, leveraging zoneless architecture for performance.
  • Styled using PrimeNG and PrimeIcons for consistent UI components.
  • Includes core Angular concepts: services, guards, HTTP calls, role-based routing, and more.
  • While not visually extravagant, the app emphasizes architectural clarity and concept coverage.

Backend Stack

  • Powered by the latest Spring Boot framework.
  • Designed using a robust microservice architecture.
  • Each service has its own database for isolation and scalability.
  • Includes Eureka Server for service discovery and Spring Cloud Config Server for centralized configuration.
  • Secured via Spring Security with role-based authentication.
  • Features an Auth Service, API Gateway, and JWT-based flows (cookie + Bearer).

App Functionality

  • Users can browse and book hotels.
  • Hotel Managers can add hotels, update rooms, and initialize availability for upcoming months etc.
  • Admin Dashboard for managing users, hotels, and system-wide configurations.

This project is a technical showcase and a portfolio piece. It reflects my passion for building scalable, secure, and maintainable full-stack applications.
